Exercices

Mettez en pratique vos connaissances avec des exercices guidés

Introduction à l'analyse de données

Un projet guidé complet couvrant les bases de l'analyse de données avec R.

✓ Importation de données ✓ Nettoyage de données ✓ Visualisation ✓ Analyse statistique
Niveau: Débutant Durée estimée: 1-3h

Nettoyage des données du CES

Un exercice pratique de nettoyage de données utilisant l'Étude électorale canadienne de 1993 et 2021.

✓ Recodage de variables ✓ Gestion des valeurs manquantes ✓ Standardisation des données ✓ Chargement et sauvegarde
Niveau: Intermédiaire Durée estimée: 2h

Analyse factorielle et régressions

Un projet complet d'analyse factorielle exploratoire suivie d'une analyse de régression, permettant d'explorer la création et l'utilisation d'échelles de mesure.

✓ Nettoyage et préparation des données ✓ Analyse factorielle exploratoire ✓ Construction d'échelles ✓ Modélisation statistique
Niveau: Avancé Durée estimée: 2-3h

Analyse de sentiment textuel

Apprenez à effectuer des analyses de sentiment sur des textes en utilisant différentes méthodes et dictionnaires.

✓ Préparation de données textuelles ✓ Analyses avec différents lexiques (AFINN, Bing, LSD) ✓ Comparaison de sources médiatiques ✓ Visualisation des résultats
Niveau: Avancé Durée estimée: 3-4h

Cours essentiels

1

Introduction to R

Apprenez les bases de R, un langage essentiel pour l'analyse de données.

2

Intermediate R

Perfectionnez vos compétences en R avec des concepts plus avancés.

3

Introduction to the Tidyverse

Découvrez l'écosystème Tidyverse pour la manipulation de données.

4

Web Scraping in R

Apprenez à extraire des données du web avec R.
5

Cluster Analysis in R

Explorez les techniques d'analyse par regroupement.
6

Introduction to Text Analysis in R

Apprenez les notions de base de l'analyse textuelle en R

Introduction to Shell
